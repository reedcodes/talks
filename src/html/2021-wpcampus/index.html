// Brand Colors
//
// Colors:
// --darkgray: rgb(37, 41, 53) - Dark Gray
// --purple: rgb(125, 59, 106) - Purple
// --cobalt: rgb(0, 121, 140) - Cobalt
// --crimson: rgb(208, 55, 17) - Crimson
// --goldenrod: rgb(246, 174, 45) - Goldenrod
// --icewhite: rgb(248, 245, 242) - Ice White
//
// Weight: 1
//
// Styleguide: Color
:root {
  --darkgray: rgb(37, 41, 53);
  --purple: rgb(125, 59, 106);
  --cobalt: rgb(0, 121, 140);
  --crimson: rgb(208, 55, 17);
  --goldenrod: rgb(246, 174, 45);
  --icewhite: rgb(248, 245, 242);
}
- First: name of the pattern - brand colors - Second: list of the colors - name, code, description - Third: weight of the page - Next: section of the styleguide, used for creating hierarchy and page URLs - After comments: regular Sass or CSS declarations 
// Link
//
// **Links** are connections between two resources...
//
// :hover - the mouse hover state
// :focus-visible - the keyboard focus state
//
// Markup: eg-link.twig
//
// Weight: 3
//
// Styleguide: Link
a {
  --color-link: var(--cobalt);
  --color-link-on: var(--purple);
  --outline-link-size: max( 2px, 0.08em );

  color: var(--color-link);

  &:hover,
  &:focus,
  &:focus-visible {
    color: var(--color-link-on);
  }

  &:focus,
  &:focus-visible {
    outline-size: var(--outline-link-size);
    outline-style: solid;
    outline-color: currentColor;
    outline-offset: var(--outline-link-size);
  }

  :focus:not(:focus-visible) { outline: none; }
}
- First: name of the pattern - Link - Second: description - Third: modifiers, or variants of the pattern, e.g. classes or pseudo-classes - Next: location of Twig to use as an example to show in the style guide - Last: section of the styleguide, used for creating hierarchy and page URLs - After comments: regular Sass or CSS declarations
